# Changelog

## v0.1.1

- Added Koopman modal decomposition via `decompose(model)`, including access to eigenvalues, modes, and eigenfunctions.
- Added support for fitting models from multiple trajectories.
- Extended monomial liftings to support exact degree ranges and explicit degree lists.
- Added `labels(...)` and `label(...)` for analytical inspection of lifted coordinates.
- Added pretty printing for liftings with input/output dimensions, delay metadata, and compact expression display.
- Expanded tests and refreshed CI workflow dependencies and architecture handling.
